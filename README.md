# Agent Skills Library

A curated collection of expert skills for AI agents in the Antigravity IDE. This repository contains structured knowledge, best practices, and code snippets designed to enhance agent capabilities across various technology stacks.

## 🚀 Purpose

This library serves as a central knowledge base for Antigravity AI agents, enabling them to:
- Adhere to project-specific architectural patterns (CQRS, Clean Architecture, DDD).
- Provide expert-level assistance for modern frameworks (.NET 10, Svelte 5, SvelteKit).
- Implement industry-standard security, testing, and deployment practices.
- Ensure consistency across different development environments.

## 📂 Skill Categories

### 🔷 .NET 10
Comprehensive skills for modern .NET development:
- **CQRS & MediatR**: Architectural patterns for clean separation of concerns.
- **EF Core 10**: Modern database access and migrations.
- **MAUI Hybrid Blazor**: Cross-platform mobile and desktop development.
- **Minimal APIs**: High-performance backend services.
- **SignalR**: Real-time communication.
- **Unit Testing**: Standardized testing patterns with xUnit.
- **Docker & DevOps**: Containerization and CI/CD pipelines.

### 🔶 TypeScript / Svelte 5
Expert knowledge for the Svelte 5 ecosystem:
- **Svelte 5 Runes**: Core reactivity patterns ($state, $derived, $effect).
- **SvelteKit 5**: Framework routing, load functions, and server actions.
- **Drizzle ORM**: Type-safe database management.
- **shadcn-svelte**: Premium UI component implementation.
- **Superforms & Zod**: Robust form management and validation.
- **Authentication**: Secure implementation with Auth.js or Lucia.
- **Animations**: Rich aesthetics and transition patterns.

## 🛠️ How to Use

Add these skill files to your Antigravity IDE configuration or point the agent to this repository to leverage the expert knowledge during development.
